# Assignment

This repository contains solutions for the assigned tasks covering **Reinforcement Learning** and **Computer Vision**.

## Contents
- **Reinforcement Learning:** DQN and PPO implementations for solving the CartPole-v1 environment, including training, evaluation, learning curves, and rendered demos.
- **Computer Vision:** A classical vision pipeline to detect a brown cardboard shoebox, estimate its orientation, and compute pixel coordinates for automated sticker placement.

Each task includes:
- Fully working scripts
- Required dependencies
- Sample inputs and outputs
- Screen recordings / visual results as required

Please refer to the individual README files inside each folder for detailed instructions on how to run and reproduce the results.
